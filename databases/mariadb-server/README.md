## mariadb
Bastille template to bootstrap MariaDB Server

## Bootstrap

```shell
bastille bootstrap https://gitlab.com/bastillebsd-templates/mariadb
```

## Usage

```shell
bastille template TARGET bastillebsd-templates/mariadb
```
