## dnsmasq
Bastille template to bootstrap dnsmasq

## Bootstrap
```shell
bastille bootstrap https://gitlab.com/bastillebsd-templates/dnsmasq
```

## Usage
```shell
bastille template TARGET bastillebsd-templates/dnsmasq
```
