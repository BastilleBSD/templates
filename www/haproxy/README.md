## haproxy
Bastille template to bootstrap HAProxy

## Bootstrap
```shell
bastille bootstrap https://gitlab.com/bastillebsd-templates/haproxy
```

## Usage
```shell
bastille template TARGET bastillebsd-templates/haproxy
```
