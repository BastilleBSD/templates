## nextcloud
Bastille template to bootstrap Nextcloud

## Bootstrap

```shell
bastille bootstrap https://gitlab.com/bastillebsd-templates/nextcloud
```

## Usage

```shell
bastille template TARGET bastillebsd-templates/nextcloud
```
