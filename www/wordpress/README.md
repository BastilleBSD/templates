# wordpress
Bastille template to bootstrap Wordpress

## Bootstrap
```shell
bastille bootstrap https://gitlab.com/bastillebsd-templates/wordpress
```

## Usage
```shell
bastille template TARGET bastillebsd-templates/wordpress
```
